Cheet sheet

  • Forever

    De code in deze functie wordt herhaaldelijk uitgevoerd.

  • Show leds

    Hiermee kan een patroon van LED's op de micro:bit worden getoond.

  • On shake

    Triggert een gebeurtenis wanneer de micro:bit wordt geschud.

  • Pick randon

    Hiermee kan een nummer op het LED-display van de micro:bit worden weergegeven.

  • Show number

    Hiermee kan een willekeurig getal binnen een opgegeven bereik worden gegenereerd.

  • Loops

    Hiermee kan een stuk code herhaaldelijk worden uitgevoerd.

  • For index

    Hiermee kan een stuk code een bepaald aantal keren worden herhaald.

  • While

    Hiermee kan een stuk code worden herhaald totdat een bepaalde voorwaarde is bereikt.

  • Repeat … times

    Hiermee kan een stuk code een bepaald aantal keren worden herhaald.

  • For element value of

    Hiermee kan een stuk code worden herhaald voor elk element in een array.

  • Set…to…

    Hiermee kan de waarde van een variabele worden ingesteld.

  • Array

    Hiermee kan een verzameling van gegevens van hetzelfde gegevenstype worden gemaakt en beheerd.

  • If

    Hiermee kan een stuk code worden uitgevoerd als een bepaalde voorwaarde waar is.

  • Else

    Hiermee kan een stuk code worden uitgevoerd als een bepaalde voorwaarde niet waar is.

  • Else-if

    Hiermee kan een extra voorwaarde worden toegevoegd die wordt gecontroleerd als de eerste voorwaarde niet waar is.

  • Variablen

    Hiermee kan een waarde worden opgeslagen en opnieuw worden gebruikt in een programma.

  • Datatypen

    Dit verwijst naar de verschillende soorten gegevens die in een programma kunnen worden gebruikt, zoals strings, floats, integers, doubles en booleans.

  • String

    Een data type dat een reeks tekens bevat.

  • Float

    Een data type dat een decimaal getal bevat.

  • Integer

    Een data type dat een geheel getal bevat.

  • Double

    Een data type dat een groot decimaal getal kan bevatten.

  • Boolean

    Een data type dat alleen de waarden "waar" of "onwaar" kan aannemen.
